# magic-8-ball-shake

A polished, mobile-friendly Magic 8 Ball web app that reveals a random answer when you tap to shake it.

## Features

- Tap or press **Shake** to animate the Magic 8 Ball
- Random classic 8 Ball-style answers
- Smooth shake motion with fade-in answer reveal
- Optional question input for extra fun
- Responsive layout for mobile and desktop
- Stylish neon glassmorphism interface

## Files

- `index.html` – app entry
- `styles.css` – layout, theming, and animations
- `app.js` – random answer logic and interaction handling

Open `index.html` in preview to use it.
